FINAL PROJECT: TEST CALCULATOR

This an Assembly LC-3 program that gets five test scores to calculate
the users average grade, minimum grade and max grade, which are displayed to the user.

The three inputs are supposed to be in the fashion of 000.
grade of A would be 100 - 099. 
grade of B would be 080 - 089.
grade of C would be 070 - 079.
grade of D would be 060 - 069.
grade of F would be 080 - 059.


TO USE THIS PROGRAM:
Copy and paste the code of the .asm file into an LC-3 simulator editor: http://lc3tutor.org/ . 
Next, assemble the file and open it (.obj file) in an LC-3 simulator and run.
